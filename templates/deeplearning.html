Tomato Leaf Disease Prediction

Tomato leaf disease prediction estimates the risk of leaf diseases in tomato plants. It helps farmers and gardeners take preventive measures for a healthier crop.

Given an input image of a tomato leaf, the goal is to determine if the leaf is affected by any of the following diseases:

  • Bacterial spot
  • Early blight
  • Leaf mold
  • Septoria leaf spot
  • Two-spotted spider mite
  • Target spot
  • Yellow leaf curl virus
  • Mosaic virus

  • Pre-trained Model: 98%

To use the tomato leaf disease application, simply upload an image of the affected tomato leaf and click "predict." The application will analyze the image and provide a prediction of the specific disease affecting the tomato plant.

Concrete Strength Prediction(Deep Learning)

The Concrete Strength Prediction model using Deep Learning is an advanced solution that accurately predicts concrete strength based on various input parameters. By utilizing deep neural networks and advanced machine learning algorithms, the model extracts meaningful representations from the data and provides precise estimates of concrete strength.

The following variables as input or predictor variables: cement, slag, fly ash, water, superplasticizer, coarse aggregate, fine aggregate, age.

ARCHITECTURE:

Hidden Layers:

  • LSTM layer with 64 nodes

Output Layer:

  • 1 node, Linear Activation Function

Loss: Mean Squared Error

Optimizer: rmsdrop

Metrics: Mean Absolute Error

ACCURACY: 97%
HOW TO USE

To use the concrete strength prediction application, input the relevant information and click "predict." The application will analyze this data and provide an estimation of the concrete's strength.
